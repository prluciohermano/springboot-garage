Corpo do Projeto

ma falha comum que vimos em muitos frameworks é a falta de suporte para texto realmente responsivo. Enquanto os elementos na página são redimensionados com fluidez, o texto ainda é redimensionado em uma base fixa. Para melhorar esse problema, para páginas com muito texto, criamos uma classe que dimensiona com fluidez o tamanho do texto e a altura da linha para otimizar a legibilidade do usuário. O comprimento da linha fica entre 45-80 caracteres e a altura da linha aumenta em telas menores.
